Strings:



1. Declaration: let str = "Hello World";
2. Methods: - length: Returns string length. - toUpperCase(): Converts to uppercase. - toLowerCase(): Converts to lowercase. - trim(): Removes whitespace. - split(): Splits into array. - join(): Joins array into string.
3. Properties: - charAt(): Returns character at index. - indexOf(): Returns index of substring. - lastIndexOf(): Returns last index of substring.

Math:



1. Constants: - Math.PI: Pi value. - Math.E: Euler's number.
2. Methods: - Math.abs(): Absolute value. - Math.ceil(): Rounds up. - Math.floor(): Rounds down. - Math.round(): Rounds to nearest integer. - Math.random(): Random number between 0 and 1. - Math.pow(): Exponentiation. - Math.sqrt(): Square root.


Numbers:



1. Types: - Integer: Whole numbers (e.g., 1, 2, 3). - Float: Decimal numbers (e.g., 3.14, -0.5). - NaN (Not a Number): Invalid or unreliable numeric results.
2. Methods: - toString(): Converts to string. - toFixed(): Formats with fixed decimal places. - toPrecision(): Formats with precision.

Arrays:



1. Declaration: let arr = [1, 2, 3];
2. Methods: - push(): Adds elements to end. - pop(): Removes last element. - shift(): Removes first element. - unshift(): Adds elements to beginning. - splice(): Removes/inserts elements. - slice(): Extracts subset. - indexOf(): Returns index of element. - includes(): Checks if element exists.
3. Properties: - length: Returns array length.

Objects:



1. Declaration: let obj = { name: "John", age: 30 };
2. Methods: - Object.keys(): Returns property names. - Object.values(): Returns property values. - Object.entries(): Returns property-value pairs.
3. Properties: - obj.property: Accesses property value. - obj["property"]: Accesses property value.
