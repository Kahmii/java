Strings

  • Declaration: let str = "Hello World";
  • length: Returns string length.
  • toUpperCase(): Converts to uppercase.
  • toLowerCase(): Converts to lowercase.
  • trim(): Removes whitespace.
  • split(): Splits into array.
  • join(): Joins array into string.
  • charAt(): Returns character at index.
  • indexOf(): Returns index of substring.
  • lastIndexOf(): Returns last index of substring.

Math

  • Math.PI: Pi value.
  • Math.E: Euler's number.
  • Math.abs(): Absolute value.
  • Math.ceil(): Rounds up.
  • Math.floor(): Rounds down.
  • Math.round(): Rounds to nearest integer.
  • Math.random(): Random number between 0 and 1.
  • Math.pow(): Exponentiation.
  • Math.sqrt(): Square root.

Numbers

  • Integer: Whole numbers (e.g., 1, 2, 3).
  • Float: Decimal numbers (e.g., 3.14, -0.5).
  • NaN (Not a Number): Invalid or unreliable numeric results.
  • toString(): Converts to string.
  • toFixed(): Formats with fixed decimal places.
  • toPrecision(): Formats with precision.

Arrays

  • Declaration: let arr = [1, 2, 3];
  • push(): Adds elements to end.
  • pop(): Removes last element.
  • shift(): Removes first element.
  • unshift(): Adds elements to beginning.
  • splice(): Removes/inserts elements.
  • slice(): Extracts subset.
  • indexOf(): Returns index of element.
  • includes(): Checks if element exists.
  • length: Returns array length.

Objects

  • Declaration: let obj = { name: "John", age: 30 };
  • Object.keys(): Returns property names.
  • Object.values(): Returns property values.
  • Object.entries(): Returns property-value pairs.
  • obj.property: Accesses property value.
  • obj["property"]: Accesses property value.
