Comparison and Logical Operators

Comparison

  • `==`(Equal to):
    Checks if two values are the same ignoring their data type
  • `===`(strict eauals to)
    Strictly checks if the two valuse are the same
  • `>`(Greater than )
    this compare two values and tell you which is greater than
  • `<`(less than)
    This comapare values and tell you which is less
  • `>=`(Greater than or equal to)`
    this checks if the value is greater than or equal to the compared value

Operators

    Adds the valuse together

    Subtracts the values from eachother
  • `/(divison):
    Divids the values from eachother
  • `*`(multiplication):
    multiplies the values by eachother
