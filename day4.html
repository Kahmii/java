Comparison and Logical Operators

Comparison Operators

  • == (Equal to): Checks if two values are the same, ignoring their data type.
  • === (Strict equals to): Strictly checks if the two values are the same.
  • > (Greater than): Compares two values and indicates which is greater.
  • < (Less than): Compares values and indicates which is less.
  • >= (Greater than or equal to): Checks if the value is greater than or equal to the compared value.
  • <= (Less than or equal to): Checks if the value is less than or equal to the compared value.

Logical Operators

  • + (Addition): Adds values together.
  • - (Subtraction): Subtracts values from each other.
  • / (Division): Divides values.
  • * (Multiplication): Multiplies values.
